## 📖 Introduction

Bee Write Back is a small journal app for the Raspberry Pi-based WriterDeck. It is built for short daily writing sessions, so you can type your thoughts and lock them away when you are done.

The goal is simple: give you a private place to write without opening an old paper journal and reading old entries by accident. If you like a calm, focused screen and a simple writing flow, this app fits that use case.

It also includes extra tools for the same device, such as a basic Claude chat client and other simple functions for the WriterDeck layout.

## 🧰 How to Use It

After you open the app, the main flow is simple:

1. Start the app.
2. Type your entry in the journal area.
3. Save the entry when you are done.
4. Lock the session or leave the app when you want to stop.
5. Return later and start a new entry.

The app is designed for short sessions, so you do not need to manage a large notebook or search through old pages. You can write one thought, save it, and step away.

## 🧭 Main Features

### ✍️ Private journaling

Write down your thoughts in a plain text space with little clutter on the screen.

### 🔒 Lock-away style flow

The app is built around the idea of writing first and closing the session after you finish. That helps keep old entries out of sight.

### 💬 Simple chat tool

The project also includes a basic Claude chat client for quick questions and text-based use.

### 🪶 Lightweight layout

The interface stays simple so it can work well on a small screen and with a keyboard.

### 🧩 WriterDeck-focused design

The app fits the WriterDeck form factor and the way people often use it: short, focused, text-first work.
